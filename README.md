Code to simulation test single-species decision rules across predator-prey scenarios using management strategy evaluation and the [Rceattle](https://github.com/grantdadams/Rceattle/tree/dev) package (dev branch) for commercial fisheries in the Bering Sea and Gulf of Alaska. Includes data simulation, assessment, and decision rules used across the US and Australia. Simulations are summarized by a variety of performance metrics.
